# Gigabit Ethernet Switch on FPGA

Welcome to the project repository for the **DTU course 34349: FPGA Design for Communication Systems**.

##  Project Overview
The primary goal of this project is to design, simulate, and implement a working Gigabit Ethernet Switch on an FPGA. 

This repository serves as the central for our group to collaborate and share files.
